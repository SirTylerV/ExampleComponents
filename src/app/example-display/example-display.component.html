Dynamic Table

This is a dynamic table that can take an array of objects and diplay them in a table using the ColumnConfigs and SortOptions.
There are 3 Inputs this component needs

  1. ColumnConfigs - This is an array that holds the column information for the table. This is where the property, header, column order, any css for the property, and if the column is sortable.
  2. Items - This is an array of objects that can be of any type. The Column config will determin the order and any css that will be applied to the property
  3. SortOptions - Holds the information for sorting and pagination
